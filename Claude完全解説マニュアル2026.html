🤖 第1章：Claude とは？— 全体像を掴む

Anthropic社が開発した次世代AIアシスタント「Claude」。安全性と有用性を両立させた設計思想で、世界中のビジネス・クリエイティブシーンで活用されています。

Claudeは、Anthropic（アンソロピック）社が開発・提供する大規模言語モデル（LLM）ベースのAIアシスタントです。Google出身のAI研究者であるDario Amodei、Daniela Amodeiらによって2021年に設立されたAnthropicは、「AIの安全性」を最重要ミッションに掲げています。Claudeはその理念を体現するプロダクトであり、単に「賢いAI」ではなく「信頼できるAI」として設計されています。

Claudeの最大の特徴は、長文の理解力、正確な指示追従、そして誠実さです。わからないことは「わからない」と答え、危険なリクエストには丁寧に断りを入れる。この一貫した姿勢が、多くのビジネスユーザーから高い信頼を得ています。

🌐 第3章：Webチャット（claude.ai）

最も手軽にClaudeを使える方法。ブラウザから claude.ai にアクセスするだけで、すぐに会話を開始できます。

🔒 claude.ai
図3-1：claude.ai のチャット画面 — 左にチャット履歴、中央に会話エリア、下部に入力欄

基本的な使い方

claude.ai にアクセスし、Googleアカウントまたはメールアドレスでサインアップするだけで利用開始できます。無料プランでも Sonnet モデルが使え、基本的な対話・文章生成・コーディング支援が可能です。

チャット画面のUIはシンプルで直感的です。左サイドバーにチャット履歴が並び、中央のメインエリアで会話が進行します。入力欄ではテキストの入力に加え、ファイルの添付、画像のアップロード、Web検索の指示なども行えます。

チャット画面の主要UI要素

💬

チャット履歴サイドバー

過去の会話を一覧で管理。検索、名前変更、削除が可能。フォルダ分類でプロジェクト別に整理。

📎

ファイル添付ボタン

PDF、Word、Excel、画像、コードファイルをドラッグ＆ドロップまたはクリックで添付。最大ファイルサイズ制限あり。

🔄

モデル切替メニュー

会話の途中でもモデルを切り替え可能。Opus / Sonnet / Haiku をワンクリックで変更。

🔍

Web検索トグル

最新情報が必要な質問では、Web検索を有効化して回答精度を高められます。

📌
モバイルアプリも利用可能：iOS / Android 向けのClaudeアプリがApp StoreおよびGoogle Playで配信されています。外出先からでもPCと同じ機能がフルに使えます。音声入力にも対応しており、ハンズフリーでの対話も可能です。

📁 第4章：プロジェクト機能

プロジェクト機能を使えば、関連するファイルやカスタム指示をあらかじめ登録し、専門的なAIアシスタントとして特化させることができます。

図4-1：プロジェクト構造 — ナレッジベースとカスタム指示をチャット間で共有

プロジェクトの作り方

claude.ai の左サイドバーから「新しいプロジェクト」をクリックすると作成画面が開きます。プロジェクト名を入力し、必要に応じてカスタム指示を設定します。カスタム指示とは「すべてのチャットに共通で適用される追加プロンプト」のことです。たとえば「回答は日本語で、敬語を使って」「Reactの最新バージョンを前提に」などの指示を登録しておけば、毎回書く手間が省けます。

ナレッジベースの活用

プロジェクトにはファイルをアップロードして「ナレッジベース」として登録できます。ここに登録したファイルは、そのプロジェクト内のすべてのチャットで参照されます。API仕様書、社内マニュアル、ブランドガイドライン、コーディング規約など、AIに理解してほしい情報を事前に登録することで、より的確な回答を得られます。

アップロードできるファイル形式は PDF、Word(.docx)、テキスト(.txt, .md)、コードファイル各種などです。1プロジェクトあたりの容量制限はプランによって異なります。

💡
活用アイデア：「法律相談プロジェクト」に契約書テンプレートを登録、「ブログ執筆プロジェクト」にブランドガイドとペルソナ情報を登録、「開発プロジェクト」にコードベースのドキュメントを登録 — など、用途別に専門AIアシスタントを作り分けられます。

🔍 第7章：Web検索 ＆ リアルタイム情報

ClaudeのWeb検索機能を使えば、学習データのカットオフ日以降の最新情報もリアルタイムで取得して回答に反映できます。

Claudeは膨大な学習データを持っていますが、それには時間的な限界（カットオフ日）があります。「今日のニュース」「最新の株価」「先週発表された新製品」などの情報は、Web検索を使わなければ正確に回答できません。

Web検索はclaude.ai のチャットで自動的に有効化される場合と、ユーザーが明示的にONにする場合があります。Claudeは質問の内容から「これは最新情報が必要だ」と判断すると、自動的にWebを検索してから回答を生成します。

Step 1 ユーザーが質問
→
Step 2 最新情報が必要？
→
Step 3 Web検索を実行
→
Step 4 検索結果を統合
→
Step 5 回答を生成

Web検索の特徴

🌐

自動判定

「現在の」「最新の」「今日の」といったキーワードを含む質問では、Claudeが自動的にWeb検索を実行します。

📚

ソース引用

Web検索を使った回答には、情報のソースURLが引用として付記されます。根拠を確認したいときに便利です。

🎯

複数ソース統合

複数のWebページから情報を収集し、一つにまとめて整理された回答を生成します。手動検索より効率的です。

⚡

リアルタイム性

ニュース、株価、天気予報、イベント情報など、常に変動する情報にアクセスできます。

⚠️
注意：Web検索結果はあくまで参考情報です。重要な意思決定の際は、必ず一次情報源を自分の目で確認してください。Claudeは検索結果の正確性を100%保証するものではありません。

🖥️ 第9章：Cowork モード

デスクトップアプリのCoworkモードは、非開発者向けの自動化ツール。ファイル操作、ドキュメント作成、タスク管理をAIが代行します。

図9-1：Cowork モードのワークフロー — 自然言語の指示からファイルを自動生成

Cowork でできること

Cowork モードは Claudeデスクトップアプリの機能で、軽量なLinux VM上でClaudeが動作します。ユーザーのコンピュータ上のフォルダにアクセスし、実際にファイルを作成・編集・管理できるのが最大の特徴です。

たとえば「このフォルダのCSVファイルを分析してExcelレポートにまとめて」「プレゼン資料を10枚のスライドで作って」「アップロードしたPDFの内容を要約してWordにまとめて」といった指示を出すと、Claudeが自動的にコードを実行してファイルを生成し、あなたのフォルダに保存します。

外部サービス連携（MCP）

Cowork モードでは MCP（Model Context Protocol）を通じて外部サービスとも連携できます。Gmail、Googleカレンダー、Notion、Slack、Google Drive、Figma、Canva などのサービスと接続し、「今週のカレンダーを確認して」「Notionのタスクリストを更新して」「Gmailの未読を要約して」といった操作が可能です。

📌
Cowork は現在リサーチプレビュー：機能は日々改善されています。フィードバックを送ることで、今後のアップデートに反映されます。デスクトップアプリの「Cowork」タブから利用できます。

🔌 第10章：API ＆ 開発者向け機能

Claude APIを使えば、自社のアプリやサービスにClaudeの知能を組み込むことができます。チャットボット、コンテンツ生成、データ分析など、あらゆるユースケースに対応。

API の基本構造

# Python SDK を使った基本的なAPI呼び出し
import anthropic

client = anthropic.Anthropic(
    api_key="your-api-key"
)

message = client.messages.create(
    model="claude-sonnet-4-6",
    max_tokens=1024,
    messages=[
        {"role": "user", "content": "日本の首都は？"}
    ]
)
print(message.content[0].text)

🔗 第11章：MCP（Model Context Protocol）

MCPは、Claudeと外部のツール・データソースを標準化された方法で接続するプロトコルです。「AIのUSBポート」とも呼ばれています。

図11-1：MCP接続マップ — Claudeから様々な外部サービスにシームレスに接続

MCPの仕組み

MCP（Model Context Protocol）は、Anthropicが提唱するオープン標準プロトコルです。AIモデルと外部のデータソース・ツールを接続するための統一インターフェースを定義しています。従来、各サービスごとに個別のAPI連携コードを書く必要がありましたが、MCPを使えば標準化された方法で「プラグイン」のように接続できます。

MCPサーバーは「ツール」「リソース」「プロンプト」の3種類の機能を公開できます。たとえばGmail MCPサーバーは「メール送信」「メール検索」「下書き作成」などのツールを提供し、Claudeはこれらのツールをユーザーの指示に応じて自動的に選択・実行します。

対応サービス例

📧

Gmail

メールの読み取り、検索、下書き作成。「未読メールを要約して」で瞬時に対応。

📅

Google Calendar

予定の確認、作成、変更。空き時間の検索や会議調整も自動化。

📝

Notion

ページの作成・更新、データベース検索、コメント追加。ナレッジ管理を効率化。

🎨

Figma / Canva

デザインファイルの読み取り、スクリーンショット取得、デザイン生成。

📌
プラグインで簡単追加：Cowork モードでは「プラグイン」として MCP サーバーをワンクリックで追加できます。マーケットプレイスから欲しいサービスのプラグインを探してインストールするだけです。

🔒 第13章：セキュリティ ＆ プライバシー

Anthropicはユーザーのデータ保護を最優先に設計。安心して業務に使える堅牢なセキュリティ体制を整えています。

データの取り扱い

Claudeとの会話データがどう扱われるかは、利用プランによって異なります。基本的な原則として、APIおよびTeam/Enterpriseプランでは、ユーザーデータはモデルのトレーニングに使用されません。Freeプランでは、サービス改善のために匿名化された形でデータが使われる可能性がありますが、オプトアウトも可能です。

図13-1：セキュリティ体制の全体像 — 暗号化・認証・コンプライアンスの三位一体

安全性への取り組み

Anthropicは「Constitutional AI」と呼ばれる独自の手法でClaudeの安全性を担保しています。これは、AIに明文化されたルール（憲法）を与え、それに基づいて自己評価・自己修正を行う仕組みです。有害なコンテンツの生成を防ぎつつ、有用な回答を最大化するバランスを追求しています。

また、ユーザーの指示が倫理的に問題のある場合は、Claudeは丁寧に断る設計になっています。「わからないことはわからない」と正直に答え、不確実な情報を確信的に述べることを避けます。この誠実さが、ビジネスシーンでの信頼につながっています。

⚠️
重要：機密情報（パスワード、APIキー、個人情報など）はチャットに入力しないようにしてください。特にFreeプランでは、データがサービス改善に使われる可能性があります。業務で機密情報を扱う場合はTeamまたはEnterpriseプランの利用を推奨します。
